Bem-vindo à documentação da API RocketSend. Aqui você encontrará todas as informações necessárias para integrar nosso serviço aos seus projetos e gerenciar as instâncias do WhatsApp de forma eficaz.

## Inicio

Para começar a usar a API RocketSend, é crucial entender como autenticar e fazer chamadas para os endpoints disponíveis.

## Autenticação

Todas as solicitações à API devem ser autenticadas usando o seguinte cabeçalho HTTP:

```bash 
    Authorization: Bearer <YOUR_API_TEAM_TOKEN>
```

## Fazendo chamadas de API

Para fazer uma chamada à API, você precisará usar o método HTTP apropriado para o endpoint desejado. Nossa API suporta os métodos ```GET```, ```POST```, ```PUT``` e ```DELETE```, dependendo da operação que você deseja realizar.
